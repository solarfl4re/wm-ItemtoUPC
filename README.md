# wm-ItemtoUPC
Takes a URL or item # from walmart.com or the mobile app (Android & iOS), and makes a UPC.

*Requirements:*
- WM Labs API key (https://developer.walmartlabs.com/member)
- JsBarcode (https://github.com/lindell/JsBarcode)
